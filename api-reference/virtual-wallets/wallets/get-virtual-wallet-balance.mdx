---
title: "Get virtual wallet balance"
description: "Retrieve token balances for a specific virtual wallet."
api: "GET /virtual-wallets/{walletId}/balance"
---

Retrieves the token balances of a specific virtual wallet by its unique identifier. Requires an API key with access level 1 or higher.

## Path parameters

<ParamField path="walletId" type="string" required>
  Unique identifier of the virtual wallet.
</ParamField>

## Response

Returns an array of balance objects.

<ResponseField name="name" type="string">
  Token name.
</ResponseField>

<ResponseField name="symbol" type="string">
  Token symbol.
</ResponseField>

<ResponseField name="balance" type="number">
  Current balance amount.
</ResponseField>

<ResponseField name="token_address" type="string">
  Smart contract address of the token.
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.cryptomate.me/virtual-wallets/vw_a1b2c3d4-e5f6-4a7b-8c9d-0e1f2a3b4c5d/balance" \
  -H "x-api-key: $CRYPTOMATE_API_KEY"
```

</RequestExample>

<ResponseExample>

```json 200 OK
[
  {
    "name": "USD Coin",
    "symbol": "USDC",
    "balance": 250.75,
    "token_address": "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48"
  }
]
```

</ResponseExample>
