---
title: "List virtual wallets"
description: "List all virtual wallets for a blockchain."
api: "GET /virtual-wallets/list"
---

Retrieves all virtual wallets for the specified blockchain. Solana is not supported. Requires an API key with access level 1 or higher.

## Query parameters

<ParamField query="blockchain" type="string" required>
  Blockchain to filter virtual wallets. One of `ETHEREUM`, `POLYGON`, `BSC`, `ARBITRUM`, `OPTIMISM`, `BASE`, `AVALANCHE`.
</ParamField>

## Response

Returns an array of virtual wallet objects.

<ResponseField name="id" type="string">
  Unique virtual wallet identifier.
</ResponseField>

<ResponseField name="alias" type="string">
  Wallet alias or display name.
</ResponseField>

<ResponseField name="client_id" type="string">
  Associated client identifier.
</ResponseField>

<ResponseField name="wallet_address" type="string">
  On-chain wallet address.
</ResponseField>

<ResponseField name="blockchain" type="string">
  Blockchain network.
</ResponseField>

<ResponseField name="enabled" type="boolean">
  Whether the wallet is enabled.
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.cryptomate.me/virtual-wallets/list?blockchain=POLYGON" \
  -H "x-api-key: $CRYPTOMATE_API_KEY"
```

</RequestExample>
