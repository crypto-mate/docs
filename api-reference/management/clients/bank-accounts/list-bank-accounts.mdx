---
title: "List bank accounts for client"
description: "List all bank accounts associated with a client."
api: "GET /management/clients/{clientId}/banks"
---

Retrieves all bank accounts associated with a specific client. Requires an API key with access level 1 or higher.

## Path parameters

<ParamField path="clientId" type="string" required>
  Unique identifier of the client.
</ParamField>

## Response

Returns an array of bank account objects. See [Create US bank account](/api-reference/management/clients/bank-accounts/create-us-bank-account) for the field list. Each account exposes rail-specific fields (e.g., `clabe`, `iban`/`bic`, `pix_key`) depending on its `type`.

<RequestExample>

```bash cURL
curl -X GET "https://api.cryptomate.me/management/clients/client_a1b2c3d4-e5f6-4a7b-8c9d-0e1f2a3b4c5d/banks" \
  -H "x-api-key: $CRYPTOMATE_API_KEY"
```

</RequestExample>
