---
title: "Get bank account"
description: "Retrieve a specific bank account."
api: "GET /management/clients/{clientId}/banks/{bankId}"
---

Retrieves detailed information about a specific bank account. Requires an API key with access level 1 or higher.

## Path parameters

<ParamField path="clientId" type="string" required>
  Unique identifier of the client.
</ParamField>

<ParamField path="bankId" type="string" required>
  Unique identifier of the bank account.
</ParamField>

## Response

See [Create US bank account](/api-reference/management/clients/bank-accounts/create-us-bank-account) for the field list.

<RequestExample>

```bash cURL
curl -X GET "https://api.cryptomate.me/management/clients/client_a1b2c3d4-e5f6-4a7b-8c9d-0e1f2a3b4c5d/banks/ba_abc123" \
  -H "x-api-key: $CRYPTOMATE_API_KEY"
```

</RequestExample>
