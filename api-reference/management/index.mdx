The Management API lets you administer your CryptoMate account: update company details and webhooks, create and manage end-clients (individual and business), link clients to external provider customer IDs, register client bank accounts across payment rails (US/ACH, SPEI, SEPA, PIX), configure payment destinations, and inspect operations, blockchains, and contracted product usage.

All endpoints are authenticated with the `x-api-key` header. Most write endpoints require an API key with access level 3; read endpoints typically require access level 1 or higher.
